# 🚀 DevDash - Developer Dashboard

A comprehensive, AI-powered personal dashboard for developers to maximize productivity, track work, and stay organized. Built with vanilla JavaScript and modern web technologies.

## ✨ Core Features

### 📊 Productivity Tracking
- **🍅 Pomodoro Timer** - Customizable focus sessions with break reminders
- **⏱️ Time Tracker** - Track time spent on different projects with detailed logs
- **✅ Smart TODO List** - AI-powered task prioritization and completion tracking
- **🔥 Habit Tracker** - Build streaks and track daily habits
- **📁 Project Manager** - Organize and track multiple projects

### 📈 Analytics & Insights
- **📊 Productivity Charts** - Visualize your progress with custom charts
- **🎯 Focus Score** - Real-time productivity scoring
- **📅 Weekly Overview** - 7-day activity heatmap
- **🧠 AI Suggestions** - Smart recommendations based on your patterns
- **📉 Time Distribution** - See where your time goes
- **🔍 Pattern Analysis** - Discover your most productive hours and days

### 🤖 AI Assistant
- **💡 Smart Task Prioritization** - AI ranks your tasks by urgency and importance
- **🎯 Productivity Insights** - Personalized tips based on your work patterns
- **⏰ Break Reminders** - Smart suggestions for when to take breaks
- **📊 Progress Predictions** - Estimate task completion times
- **🌟 Motivational Messages** - Context-aware encouragement

### 🌐 Integrations
- **📊 GitHub Stats** - Comprehensive GitHub profile analytics
- **🌤️ Weather Widget** - Real-time weather for your location
- **💰 Crypto Prices** - Live cryptocurrency tracking
- **📰 Dev News** - Latest Hacker News stories
- **❓ Stack Overflow** - Trending questions in your tech stack

### 💻 Developer Tools
- **💻 Code Snippets** - Save and organize code with syntax highlighting
- **🔖 Bookmarks Manager** - Categorized bookmark organization
- **🔗 Quick Links** - Fast access to dev resources (GitHub, MDN, DevDocs, etc.)
- **📝 Quick Notes** - Persistent notepad with auto-save

### 📅 Organization
- **📅 Calendar View** - Visual calendar with event management
- **⏰ Reminders** - Scheduled notifications with repeat options
- **🎯 Goals** - Daily, weekly, and monthly goal tracking
- **📊 Activity Heatmap** - 12-week productivity visualization

- **Vanilla JavaScript** - 4200+ lines of pure JS, no frameworks
- **Tailwind CSS** - Modern, responsive glassmorphism design
- **LocalStorage** - Client-side data persistence
- **Canvas API** - Custom charts and visualizations
- **Web APIs** - Notifications, Vibration, Battery, Network, Share

## 🚀 Getting Started

### Quick Start
1. Clone or download this repository
2. Open `index.html` in your browser
3. Click the settings icon (⚙️) to configure:
   - GitHub username
   - Pomodoro durations
   - Theme preferences
4. Start tracking your productivity!

### Installation as PWA
1. Open DevDash in Chrome/Edge/Safari
2. Click "Install" when prompted
3. Access from home screen like a native app

**No build process, no npm install, no dependencies!**

## 📖 Usage Guide

### ⌨️ Keyboard Shortcuts
- `Ctrl+K` - Open command palette
- `Ctrl+N` - New task
- `Ctrl+S` - Save notes
- `Ctrl+H` - New habit
- `Ctrl+P` - New project
- `Ctrl+T` - New snippet
- `Ctrl+,` - Settings
- `Space` - Start/pause Pomodoro (when not typing)
- `Escape` - Close modals

### 🍅 Pomodoro Timer
1. Click "Start" to begin a 25-minute focus session
2. Work without distractions
3. Take a 5-minute break when timer ends
4. After 4 sessions, take a longer 15-minute break
5. Customize durations in settings

### 🔥 Habit Tracker
1. Add habits you want to build
2. Check them off daily
3. Build streaks to stay motivated
4. Track your consistency over time

### 📊 Analytics
- View daily productivity stats
- See weekly trends and patterns
- Discover your most productive times
- Get AI-powered insights and suggestions

### 📱 Mobile Features
- Pull down to refresh
- Swipe gestures for navigation
- Install as PWA for offline access
- Touch-optimized interface
- Vibration feedback

### 💾 Data Management
- **Export**: Backup your data as JSON
- **Import**: Restore from backup
- **Auto-save**: Changes saved automatically
- **Privacy**: All data stays on your device

## 🎨 Customization

The app uses a purple gradient background with glassmorphism effects. To customize:

- Edit the `body` gradient in `index.html` `<style>` section
- Modify Tailwind classes for different colors
- Adjust the `.glass` effect for different transparency
